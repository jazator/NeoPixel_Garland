# NeoPixel_Garland
Эффекты для гирлянды на основе библиотеки NeoPixel.
 
Функция `OneColor` заливает всю ленту одним цветом с заданной скоростью.

### Параметры
`OneColor(R, G, B, second)`

- `R, G, B` — компоненты цвета (0..255).
- `second` — скорость заливки в миллисекундах на один светодиод:
  - `0` — мгновенная заливка всей ленты;
  - `> 0` — заливка от начала ленты к концу;
  - `< 0` — заливка от конца ленты к началу.

### Пример использования
В файле `OneColor.ino` уже есть рабочий пример с тремя последовательными заливками.
Можно добавить любое количество вызовов `OneColor(R, G, B, second)` — они будут выполняться по порядку.
